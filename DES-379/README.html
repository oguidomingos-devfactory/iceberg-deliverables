📋 Pulso Google Ads Preview Template

Template HTML reutilizável que simula a página de resultados de busca do Google com um anúncio patrocinado (Google Ads), customizável para qualquer especialidade médica.

✅ Template pronto! Use os parâmetros abaixo para personalizar o anúncio.

🎯 Objetivo

Quando um lead (médico/clínica) entra no funil da Pulso, usar este template para mostrar como ficaria o anúncio dele no Google Search. Ferramenta de vendas para fechar contratos de tráfego pago.

🚀 Como Usar

URL Base

https://jarvis.icebergcompany.com.br/deliverables/DES-379/index.html

Parâmetros de Query String

keyword
Palavra-chave buscada no Google (ex: "ginecologista perto de mim")
headline
Headline do anúncio (ex: "Dra. Maria Silva | Ginecologista - Agende sua Consulta")
description
Descrição do anúncio com diferenciais (ex: "Mais de 15 anos de experiência. Atendimento humanizado. Agende online.")
url
URL de display (ex: "www.doutora-maria.com.br")
phone
Extensão de telefone (ex: "(11) 98765-4321")
location
Localização (ex: "São Paulo, SP")
sitelink1, sitelink2, sitelink3, sitelink4
Links adicionais (ex: "Agendar Consulta", "Localização", "Sobre a Dra.", "Convênios")

💡 Exemplos de Uso

Exemplo 1: Ginecologista

URL:
index.html?keyword=ginecologista%20perto%20de%20mim&headline=Dra.%20Maria%20Silva%20%7C%20Ginecologista&description=15%20anos%20experiência.%20Humanizado.%20Agende%20online.&url=www.doutora-maria.com.br&phone=%2811%29%2098765-4321&location=São%20Paulo,%20SP&sitelink1=Agendar&sitelink2=Localização&sitelink3=Sobre&sitelink4=Convênios

Exemplo 2: Dermatologista

URL:
index.html?keyword=dermatologista%20em%20são%20paulo&headline=Dr.%20João%20Santos%20%7C%20Dermatologista&description=Estética%20e%20clínica.%20Tecnologia%20laser.%20Consultas%20online.&url=www.dermatologia-joao.com.br&phone=%2811%29%2099999-8888&location=São%20Paulo,%20SP&sitelink1=Marcar&sitelink2=Procedimentos&sitelink3=Localização&sitelink4=Avaliação%20Grátis

📱 Características

  • Fidelidade ao Google Search — Layout, cores, tipografia e espaçamento idênticos ao real
  • Badge "Patrocinado" — Identificação clara do anúncio
  • Extensões de Anúncio — Telefone, localização, sitelinks (como Google Ads real)
  • Resultados Orgânicos — Contexto realista com 3 resultados abaixo do anúncio
  • Responsivo — Desktop + Mobile optimizado
  • Parametrizado — Customize via query string para qualquer especialidade
  • Sem dependências externas — HTML + CSS puro, rápido, offline-ready

🎨 Design Notes

  • Cores seguem paleta oficial Google (azul #1a0dff para links, verde #006621 para URLs)
  • Tipografia system fonts (SF Pro, Segoe UI) para fidelidade
  • Espaçamento e tamanhos de fonte baseados em padrões Google
  • Totalmente acessível — contraste WCAG AA, semântica HTML
